Depois — sistema RAYO aplicado

Canvas areia, marca terracota, voz editorial.

Mesma família visual do produto: forest como ink, terracota como CTA, sage como respiro, Outfit em pesos mistos.

  • Canvas #FAF4E8 / borda #D9C89E
  • CTA terracota #C8553D · pill
  • Bullets com traço (—) em vez de disc
  • Brand mark do RAYO (símbolo do sol) + wordmark
  • Outfit 600 + 200 italic para ênfases

Tokens

02 / 07

Use estes valores hexadecimais literais nos inline styles dos emails. Não invente novos.

Surfaces
canvas#FAF4E8
backdrop#F2E9D5
border#D9C89E

canvas é o fundo de TODO email (corpo da mensagem). backdrop é a "moldura" que aparece atrás em clientes desktop. border separa seções e contorna cards.

Brand · CTA
terra-500#C8553D
terra-700#9C3A26
forest-900#0C3B2E

terra-500 é a cor do botão primário (sempre). forest-900 é a tinta principal de texto. terra-700 é hover/dark variant.

Accents
sage-500#6D9773
sage-100#DCE8D2
ochre-500#D4A24C

sage para confirmações positivas (sucesso, completou trilha). ochre para pequenos highlights e badges premium. Use com parcimônia.

Ink (texto)
ink-900#0E1A14
ink-700#1F2A22
ink-500#4A5247
ink-400#6E7569

ink-900 = headlines. ink-700 = corpo. ink-500 = legendas. ink-400 = footer/disclaimers.

Geometry
radius card14px
radius CTA999px (pill)
radius mark10px
container600px
padding ext48px / 32px (mob)

600px é o padrão de email — não ultrapassar. Padding externo (48px desktop / 32px mobile) garante respiração.

Type stack
Outfit · webfont
Helvetica Neue · fallback
Arial · final
sans-serif · safe

Stack completo: 'Outfit','Helvetica Neue',Arial,sans-serif. Importar Outfit pela tag <link> Google Fonts no head.

Especificação técnica

04 / 07

Regras absolutas que valem para qualquer email — transacional ou marketing.

Largura 600px de largura útil. Centralizar na viewport usando <table> outer com width="100%".
Backdrop Fora do container: #F2E9D5 (sand-100). Dentro do container: #FAF4E8 (sand-50). Não usar fundos escuros.
Container Card central com border: 1px solid #D9C89E + border-radius: 14px + padding: 48px 40px (32px mobile).
Header Brand mark (sol terracota 34×34, radius 10) + wordmark "RAYO" (Outfit 700, 18px, -0.04em). Esquerda do container, sem badge/pill em volta.
Eyebrow Sempre presente acima do título. Monospace, 11px, +0.22em letter-spacing, uppercase, cor terracota, com traço de 24×1px antes do texto.
Headline 28–38px. Mistura peso 600 (forest-900) + peso 200 italic (terra-500). Quebra de linha intencional. Saudação personalizada quando possível.
CTA Botão único background:#C8553D + color:#FAF4E8 + border-radius:999px + padding:16px 32px. Texto verbo+objeto ("Acessar plataforma", "Continuar trilha"). Nunca múltiplos CTAs primários no mesmo email.
Bullets Lista sem disc. Cada item com traço terracota de 14×1.5px antes (não usar •). Espaçamento gap:10px.
Footer Fora do card (no backdrop). Logo pequeno + endereço + link de descadastro. Cor ink-400. Mono 11px para metadata.
Tabela > div Use <table> com role="presentation" para layout — clientes Outlook não suportam flexbox/grid. Inline styles em style="", não classes CSS para os blocos críticos.
Imagens Sempre com alt="" descritivo. Bloco-padrão (capa, ilustração): 520×260, radius 10px. Hospedar em CDN, nunca embed base64.
Modo escuro Gmail/Apple Mail invertem cores automaticamente. Não usar meta color-scheme — deixar o cliente decidir. Garantir contraste com forest-900/sand-50 que funcionam em ambos.

Voz e tom

07 / 07

Regras curtas que a IA deve seguir ao escrever o copy.

Pessoa Segunda pessoa do singular (você), nunca "Sr./Sra.". Saudar pelo nome sempre que possível.
Postura Acolhedor, sereno, editorial. Não vendedor. Não corporativo. Não emoji-feliz.
Frases Curtas. Verbos no presente. Evitar "transformação", "jornada épica", "revolucionar". Preferir "trilha", "ritmo", "passo", "prática".
CTA copy Verbo + objeto. "Acessar plataforma", "Continuar trilha", "Ler carta". Nunca "Clique aqui", "Saiba mais".
Emoji Não usar.
Assunto Máximo 50 caracteres. Concreto e específico. Ex: "Sua trilha de comunicação te espera" — não "Olá! Veja o que preparamos para você 🌟".
