## 🎨 ESPECIFICAÇÕES DE DESIGN

### Tipografia:
```
Font Family: Urbanist
Weights disponíveis:
- Regular (400) - Não usar
- Medium (500) - Não usar
- SemiBold (600) - Subtextos
- Bold (700) - Títulos principais
- ExtraBold (800) - Headlines hero

Tamanhos:
- Hero (Screenshot 1): 48-52pt
- Headlines: 40-44pt
- Subtexts: 28-32pt
- Disclaimers: 18-22pt
```

### Cores:

**Texto Principal:**
```css
Branco: #FFFFFF
Sombra: rgba(0, 0, 0, 0.4)
Stroke: 2px branco (para contraste)
```

**Texto Accent:**
```css
Amarelo RAIO: #D97706 (light mode)
Amarelo RAIO: #FBBF24 (dark mode)
```

**Background Overlays:**
```css
Escuro: linear-gradient(180deg, rgba(0,0,0,0.6) 0%, rgba(0,0,0,0) 100%)
Claro: linear-gradient(180deg, rgba(255,255,255,0.8) 0%, rgba(255,255,255,0) 100%)
```

### Layout:

**Safe Zones:**
```
Top: 120px (status bar + notch)
Bottom: 160px (home indicator + padding)
Sides: 40px (breathing room)
```

**Alinhamento:**
```
Títulos: Centro ou esquerda
Subtextos: Sempre centro
CTAs: Centro
Badges/icons: Canto superior direito
```

## 💡 DICAS DE OURO

### DO's ✅
- Foque em BENEFÍCIOS, não features
- Use verbos de ação
- Seja específico ("50+ cursos" > "muitos cursos")
- Crie urgência sutil
- Mostre social proof
- Mantenha consistência visual

### DON'Ts ❌
- Não use mais de 10 palavras
- Não seja genérico
- Não exagere nas promessas
- Não use jargão técnico
- Não copie concorrentes
- Não use pontuação excessiva (!!!)
