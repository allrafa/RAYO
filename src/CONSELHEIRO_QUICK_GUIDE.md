## ⌨️ ATALHOS

### **Teclado:**

- **Enter** = Enviar mensagem
- **Esc** = Fechar menu de sugestões

### **Gestures:**

- **Scroll ↓** = Esconde header + bottom bar
- **Scroll ↑** = Mostra header + bottom bar
- **Tap em [ⓘ]** = Toggle disclaimer

## 🔧 TROUBLESHOOTING

### **Header não some ao rolar:**

```
→ Verifique se está rolando pelo menos 50px
→ Header só some após passar do topo
```

### **Chips não somem:**

```
→ Envie primeira mensagem
→ Chips somem automaticamente após envio
```

### **Disclaimer não aparece:**

```
→ Hover sobre ícone [ⓘ] (desktop)
→ Tap no ícone [ⓘ] (mobile)
```

### **Bottom bar não reaparece:**

```
→ Role para cima
→ Bottom bar reaparece automaticamente
```
