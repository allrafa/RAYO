## 🎮 PRINCÍPIOS CORE

### Filosofia "Sem Punição"
```
✅ Energia regenera (nunca bloqueia)
✅ Streaks protegidos (Freeze, Weekend Pass)
✅ Missões opcionais (não obrigatórias)
✅ Celebrações frequentes
✅ Feedback positivo sempre

❌ NUNCA bloquear conteúdo
❌ NUNCA criar ansiedade
❌ NUNCA comparações negativas
❌ NUNCA punir por inatividade
```

### Inspirações
- **Duolingo:** Streaks + Energia (mas melhorado - sem bloqueio!)
- **Habitica:** Missões + Recompensas (mas com conteúdo real)
- **Peloton:** Streaks sociais (mas assíncrono e flexível)
- **Nir Eyal:** Hooked model (mas ético e transparente)
